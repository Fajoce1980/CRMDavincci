# CRMDavincci
Prueba Tecnica para la empresa Davincci
Base de Datos sql. server
WinForms C#
Clase CRMBD contiene la cadena de conexion hacia la base de Datos
La clase CampañsController: Contiene los metodos para Cargar y Leer Archivos CSV y txt, cuya informción este separada por delimitaores como: ,,;,-, espacios en blanco o cualquier otro caracter; Metodos para Buscar el codigo de campaña asignado a un cliente especifico; Ver la informacion que se guarda en la tabla de Base de Datos y Guardar las columnas especificas de los archivos cargados
